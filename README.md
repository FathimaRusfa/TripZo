# TripZo Tourist Guide System

## Description
TripZo is a PHP/MySQL web application designed to help users discover attractions, explore locations via an interactive map, and plan trips in Addalaichenai.

## Features
- Attraction browsing
- Interactive map (Leaflet)
- Trip planner
- Reviews system
- Admin dashboard
- Multi-image gallery

## Technologies
- PHP
- MySQL
- Bootstrap 5
- Leaflet.js

## Setup
1. Import tripzo.sql
2. Configure db.php
3. Run on XAMPP or hosting
